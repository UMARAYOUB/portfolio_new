Diabetes Prediction System

"HealthGuard: Diabetes Prediction System," is an AI-based tool that predicts diabetes risk using a Random Forest Classifier and an expert system (Experta) for reasoning. The system features a Tkinter-based GUI for user interaction and provides explainable results. It utilizes the Pima Indians Diabetes dataset, achieving 72% accuracy in predictions. The project ensures user-friendly interaction, accurate risk assessment, and interpretability, with future potential for enhancements like cloud deployment and lifestyle factor integration.

Movie Recommendation System

An AI-powered system that suggests movies based on user preferences using content-based filtering. It analyzes movie genres with TF-IDF Vectorization and cosine similarity to provide accurate recommendations. Built with Streamlit, it offers an interactive interface for seamless user experience.

Emotion Detection Using Machine Learning and Deep Learning on the ISEAR Dataset

This project focuses on emotion detection using machine learning models on the ISEAR dataset. Various traditional, deep learning, and hybrid models were tested, with LSTM + RoBERTa achieving the highest accuracy of 88%. The study highlights the effectiveness of deep learning and transformer-based models for emotion classification. Future improvements may include using larger pre-trained models and real-world applications in sentiment analysis and mental health monitoring.
